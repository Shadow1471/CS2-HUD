# CS2-HUD
This HUD is inspired by the new Counter Strike 2 HUD.

# FEATURES
- Money [Text]
- Health [Bar + Text]
- Armor [Bar + Text]
- Ammo [Text]
- Player Steam Avatar
- Customizable (config file)

# CONFIGURATION
Configuration file available, if you wish to change colors, etc., go to: lua\autorun\cs2hud\shared\sh_config.lua
